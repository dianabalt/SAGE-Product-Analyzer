# Extension Icons

Please add the following icon files to this directory:

- **icon-16.png** - 16x16px icon for toolbar
- **icon-48.png** - 48x48px icon for extension management
- **icon-128.png** - 128x128px icon for Chrome Web Store

## Icon Design Guidelines

- Use sage green colors (#6B8E23 primary)
- Include a checkmark or leaf symbol
- Simple, recognizable design at small sizes
- Transparent background recommended

## Tools to Create Icons

- Figma: https://figma.com
- Canva: https://canva.com
- Adobe Illustrator
- Or use online icon generators

## Temporary Placeholders

For now, you can use colored squares as placeholders until custom icons are created.
